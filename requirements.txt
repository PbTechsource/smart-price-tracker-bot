# 🤖 Price Tracker Bot — ربات ردیاب قیمت

ربات تلگرام برای ردیابی قیمت محصولات از **دیجی‌کالا** و **دیوار** با تحلیل هوشمند ML.

---

## 🛠️ نصب و راه‌اندازی

### ۱. نصب وابستگی‌ها
```bash
pip install -r requirements.txt
playwright install chromium
```

### ۲. گرفتن توکن از BotFather
1. در تلگرام سرچ کن: `@BotFather`
2. بزن `/newbot`
3. اسم و username بات رو بنویس
4. توکن رو کپی کن

### ۳. تنظیم توکن
```bash
# Linux/Mac
export TELEGRAM_BOT_TOKEN="توکن_شما_اینجا"

# Windows
set TELEGRAM_BOT_TOKEN=توکن_شما_اینجا
```

### ۴. اجرا
```bash
python bot.py
```

---

## 📋 دستورات ربات

| دستور | توضیح |
|-------|--------|
| `/start` | خوش‌امدگویی و راهنما |
| `/add_product` | افزودن محصول جدید (لینک + نام) |
| `/products` | نمایش لیست محصولات |
| `/track <id>` | بررسی قیمت فعلی و ذخیره در دیتابیس |
| `/ml <id>` | تحلیل هوشمند و سیگنال خرید |

---

## 🧠 منطق ML

بر اساس **میانگین** و **انحراف معیار** قیمت‌های تاریخی:

- 🟢 **زمان خرید** — قیمت زیر `میانگین - ۰.۵×std` یا نزدیک کمترین قیمت
- 🔴 **هنوز نخر** — قیمت بالای `میانگین + ۰.۵×std` یا روند صعودی
- 🟡 **معقوله** — قیمت در محدوده نرمال

> ⚠️ حداقل ۳ بار track کن تا تحلیل دقیق داشته باشی!

    └── analyzer.py     # تحلیل قیمت با آمار
