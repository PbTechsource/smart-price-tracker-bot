# 🚀 Overview

Smart Price Tracker is an AI-powered Telegram bot developed by **Tech Source**.

The system automatically tracks product prices, stores historical data, and uses Machine Learning algorithms to provide intelligent buying recommendations and future price predictions.

---

# ✨ Features

- 🤖 Telegram Bot Integration
- 📦 Product Management

- 📈 Price History Database
- 🧠 Machine Learning Prediction
- 🔥 Smart Buy/Wait Recommendation

# 📋 Commands

| Command | Description |
|----------|------------|
| `/start` | Start the bot |
| `/add_product` | Add a new product |
| `/products` | Show saved products |
| `/track <id>` | Update current price |
| `/ml <id>` | Generate AI recommendation |

---

# 🧠 Machine Learning

The model analyzes:

- Current Price
- 7-Day Average Price
- 30-Day Average Price
- Lowest Price In Last 30 Days
- Percentage Price Change

Possible Outputs:

- ✅ Buy Now
- ⏳ Wait
